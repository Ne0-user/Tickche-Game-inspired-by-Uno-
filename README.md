Tickche

Tickche is a small interactive project built with Pygame that showcases the practical use of classical data structures inside a visual and dynamic environment.
The project includes custom-designed cards and uses several manually implemented structures to manage game behavior and card flow.

Project Description

Tickche is a inspired game with the porpuse to learn and apply data structures:

*LinkedList
